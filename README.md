# Procurement Spend & Supplier Performance Dashboard
This project focuses on procurement analytics and supplier performance monitoring using Microsoft Excel dashboards. The dashboard provides insights into procurement spend, payment status, supplier contribution, contract activity, and delivery performance through interactive KPI reporting and slicers.

Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- KPI Cards
- Conditional Formatting

Dataset Information
- 500+ procurement transaction records
- 10 suppliers
- 6 procurement categories
- Procurement spend exceeding ₹38.6M

Key Features
- Interactive procurement dashboard
- Supplier spend analysis
- Payment status tracking
- Contract activity monitoring
- Delivery delay analysis
- Dynamic filtering using slicers

Key Insights
- Packaging category recorded the highest procurement spend (~₹7.17M)
- ABC Packaging, Nova Industries, and Delta Packaging were the top suppliers by spend
- 156 delayed payment transactions were identified
- Average supplier delivery delay was approximately 7 days

Recommendations
- Monitor high-spend suppliers periodically for cost optimization opportunities
- Track overdue payments regularly to reduce operational delays
- Improve monitoring of supplier delivery performance
